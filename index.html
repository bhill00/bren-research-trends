This report was generated by a Retrieval-Augmented Generation (RAG) pipeline built on 623 project deliverables — final reports, proposals, executive summaries, briefs, and technical documentation — from the Bren School's MESM, MEDS, Eco-E, and Communications Focus programs.

The pipeline uses HyDE (Hypothetical Document Embeddings) to generate an ideal answer before retrieval, improving semantic matching for broad trend questions. Retrieved candidates are then reranked using a local cross-encoder model (ms-marco-MiniLM) to surface the most relevant passages. Finally, GPT-4o synthesizes answers from the top 30 reranked chunks with full source citations.
